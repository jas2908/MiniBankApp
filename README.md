# MiniBankApp

MiniBankApp is a simple Android banking app prototype built using Kotlin and Jetpack Compose.

This project demonstrates modern Android development practices including:
- Jetpack Compose UI
- ViewModel architecture
- StateFlow for state management
- Navigation between screens
- Loading & error state handling

---

## Features

- Login screen with basic validation
- Dashboard screen displaying:
  - Account balance
  - List of recent transactions
  - Loading indicator simulation
- Clean separation between UI and business logic

---

## Tech Stack

- Kotlin
- Jetpack Compose
- ViewModel
- StateFlow
- Material 3
- Compose Navigation
